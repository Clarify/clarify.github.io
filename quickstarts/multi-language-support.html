Getting Started with Multi-Language Media Search

This quickstart demonstrates a simple way to get started using the Clarify API. Following these steps, it should take you no more than 5-10 minutes to support search in a variety of languages.

We currently support English (default), Spanish (es), and French (fr). More languages are on the way.

Note: Please contact us at support@clarify.io to add languages beyond English.
