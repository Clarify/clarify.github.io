Getting your Spoken keywords

This quickstart demonstrates how to load an audio file and request your spoken keywords in just a matter of minutes.

Configuring Your Environment

While you can use any programming language you choose, we provide a few helper libraries to complete processing. In most cases, you can use your favorite package manager:

  • PHP
  • Java
  • Python
  • Ruby
  • curl

In PHP, you simply install the Clarify SDK using Composer:

And then simply run:

composer install

In Java, you simply install the Clarify SDK via your Maven pom.xml:

In Python, you simply install the Clarify module via pip:

In Ruby, you simply install the Clarify gem and set your API key as an environment variable:

Although we don’t have a curl library, the command-line JSON parser “jq” is super helpful. Download it and install it to get started: http://stedolan.github.io/jq/

Getting your Keywords

The most important part of the bundle notification is the ‘href’ of the ‘clarify:insights’ key. By retrieving the contents of that URI, Clarify will give you a list of the Insights available.

The next step is to use the ‘insight:spoken_keywords’ URI to retrieve the results of the processing.

From here, we can process the keywords just like any other JSON array. You may want to use them to perform informed searches, display word clouds, or tag your media. At this point, it’s up to you.
