These quickstarts are designed to get you started with the Clarify API quickly. With the exception of Human Transcripts, any of these quickstarts should take you 5-10 minutes to fully implement.

Media Search

See the Code »

The Media Search Quickstart gives you code to get started. The code samples walk you through getting a helper library, loading your first audio file, searching for a few keywords, and exploring the results.

Multi-language Support

See the Code »

The Multi-Language Search Quickstart demonstrates how to get started with multiple languages. The included code walks you through getting a helper library, loading a French language file, searching for a French keywords, and exploring the results.

Human Transcripts

See the Code »

The Human Transcript Quickstart demonstrates how to take any English language audio and get a transcript. The code samples walk you through getting a helper library, loading your audio file, requesting the transcript, and receiving the results.

Keywords Insight

See the Code »

The Keyword Insight Quickstart demonstrates how to quickly extract the most frequently said words from any audio or video file. The code samples walk you through getting a helper library, loading your audio file, requesting the keywords, and receiving the results.
