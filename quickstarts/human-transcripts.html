Using the Transcription API

This quickstart demonstrates how to load an audio file and request a transcript in just a matter of minutes.

Note: Since these transcripts are generated by humans, it can take upwards of 36 hours to receive the
                    callback. If you need results faster, please contact us at support@clarify.io.
