--- layout: quickstart title: Spoken Topics navtitle: Spoken Topics languages: - curl order: 5 --- The Spoken Topics insight contains track-level data. track_data is an array containing an object for each track. Each object contains the track_id and track_label of the track, as well as the following:

topics

Topics are the subjects spoken about in the audio.

The topics array contains objects with:
  • categories - Subjects that describe the topic. A category is represented by an array of path arrays which contain objects, each with the name of a subject, from most general to most specific.
  • terms - Array of terms where each is a word related to the topic. Each term is an object with a term property.
