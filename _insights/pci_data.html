--- layout: quickstart title: PCI Data navtitle: PCI Data status: invite languages: - curl order: 10 --- The PCI Data insight contains track-level data. track_data is an array containing an object for each track. Each object contains the track_id and track_label of the track, as well as the following:
  • pci_data - Sections of audio that contain credit card numbers, security codes, and any number associated with an expiry date. It is an array that contains objects with start and end times in seconds.
