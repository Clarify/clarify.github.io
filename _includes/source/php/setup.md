You can install the Clarify SDK using <a href="https://getcomposer.org/" target="_new">Composer</a>:

{% highlight javascript %}
{
    "require": {
        "clarify/clarify-helper": "~2.0"
    }
}
{% endhighlight %}

And then simply run:

{% highlight bash %}
composer install

# Don't forget to use sudo if appropriate.
{% endhighlight %}
